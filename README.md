# SmartSaver

## Smart Price Comparison Platform

SmartSaver is a full-stack web application designed to help users compare prices, ratings, offers, and features across multiple platforms for Products, Hotels, and Flights.

The goal of SmartSaver is to save users time and money by providing all comparison information in one place through a modern and user-friendly interface.

---

# Project Objectives

- Compare product prices from multiple platforms
- Compare hotel prices and facilities
- Compare flight fares and schedules
- Provide search and filtering functionality
- Offer a responsive and attractive user experience
- Implement secure user authentication
- Store and manage data using a relational database

# Installation Guide

## Clone Repository

```bash
git clone https://github.com/birralaxman36/SmartSaver---Price-Comparison-Website.git
```

## Frontend Setup

1. Open project folder
2. Run `index.html`

## Backend Setup

1. Install Java 21+
2. Install MySQL Server
3. Create `smartsaver` database
4. Configure `application.properties`
5. Run:

```bash
./mvnw spring-boot:run
```
